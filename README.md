# Event Check-In and Attendance System (JavaScript)

## Description
This project is a simple Event Check-In and Attendance System built using Node.js.

The system allows:
- Creating events
- Registering attendees using a unique email address
- Checking in registered attendees
- Generating an attendance report showing:
  - Event name
  - Total registered attendees
  - Total checked-in attendees
  - List of checked-in attendees

This project is created to demonstrate:
- Proper separation of logic and data handling
- Unit testing of business rules
- Integration testing using a real database
- Continuous Integration using GitHub Actions

---

## Requirements
To run this project, you need:
- Node.js (version 18 or higher)
- npm (comes with Node.js)
